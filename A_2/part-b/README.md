## Part B: MongoDB Client Connector with Node.js

1. Develop Node.js code to connect with bookstoreDB using MongoDB Client.
2. Create a function to insert a new book with fields (title, author, price, stock).
3. Implement a function to insert a new customer into the customers collection.
4. Design a function to insert a new order in the orders collection.
5. Construct code to fetch all books and display them in the console.
6. Develop code to fetch all customers from the customers collection.
7. Build a query to find one book by its title.
8. Implement a function to update the price of a book.
9. Create a function to update the stock quantity of a book.
10. Design code to delete a book with a specific title.
11. Develop a function to delete a customer by email.
12. Construct a query to find all books written by a specific author.
13. Implement code to count total number of documents in books.
14. Create a query to find all orders of a given customer.
15. Build a function to insert multiple books at once.
16. Design code to update multiple customer addresses at once.
17. Develop a function to display only book titles and prices (projection).
18. Implement a function to sort all books by price in descending order.
19. Construct code to find all books within a price range.
20. Create a reusable module that connects to MongoDB and exports the connection object.

## Part C: Mongoose Connector (Schemas, Models & CRUD)

1. Design a Mongoose schema for Book with fields: title, author, price, stock.
2. Create a schema for Customer with fields: name, email, address.
3. Develop a schema for Order referencing Customer and Book.
4. Implement a Mongoose model for Book.
5. Build a Mongoose model for Customer.
6. Create a Mongoose model for Order.
7. Construct code to insert a new customer into the database.
8. Develop code to insert a new book into the database.
9. Design a function to create a new order for a given customer.
10. Implement a query using populate() to fetch all orders with customer details.
11. Build a query using populate() to fetch all orders with book details.
12. Create a function to update the stock of a book when an order is placed.
13. Develop code to update a customer’s email address.
14. Construct a query to delete a customer by ID.
15. Implement a function to delete a book by ID.
16. Build a query to list all books priced above 40.
17. Design code to fetch all customers living in “Lahore”.
18. Develop validation rules in the Book schema (e.g., price must be > 0).
19. Create a custom method in the Customer schema to display full info.
20. Construct a virtual field in Order schema that calculates order total dynamically.
